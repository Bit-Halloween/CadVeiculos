Sistema de cadastro de veículos em Java.

## O que o programa faz

O **CadVeiculos** é um programa de console que permite gerenciar uma lista de veículos através de um menu interativo. 

### Funcionalidades

- Cadastrar novo veículo (com verificação de duplicatas)
- Listar todos os veículos (ordenados alfabeticamente)
- Remover veículo por índice
- Buscar veículo por nome
- Editar veículo por índice
- Remover veículo por nome
- Sair do programa
